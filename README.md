# Rummelbude

## Über mich
Ich bin ein leidenschaftlicher Einzelkünstler, der seine Musik kostenlos und ohne Werbung veröffentlicht. Mein Ziel auf GitHub ist es, eine neue Website aufzubauen, die
[die aktuelle Website](https://rummelbude.wixsite.com/start) ersetzt und mithilfe von [GitHub Pages](https://github.io) veröffentlicht wird.

Unterstütze das Projekt jetzt! Schau dir [die neue Website](https://rummelbude.github.io/website) an und erstelle Pull requests im [Repository](https://github.com/Rummelbude/website), wenn du Verbesserungsvorschläge hast.
